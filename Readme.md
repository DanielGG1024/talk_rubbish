## Environment SetUp

    node: v10.15.0
    nodemon 2.0.7
    express: ^4.17.1
    express-handlebars: ^5.3.2

## Installing 

1. clone 此專案
```
git clone https://github.com/DanielGG1024/talk_rubbish.git
```
2. 使用終端機移動至該目錄
```
cd talk_rubbish
```
3. 安裝npm 套件
```
npm install
```
4. 啟動伺服器
```
npm run dev
```
5. 出現'express server is running on http://localhost:3000'
表示成功
